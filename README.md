# 🌍 CrisisMap AI

CrisisMap AI is an AI-powered disaster management platform that enables users to report emergencies, visualize incidents on an interactive map, and receive real-time alerts. The system helps authorities monitor disasters, prioritize responses, and improve crisis management using AI-driven analytics.

## 🚀 Features
- User authentication (JWT)
- Disaster reporting with location
- Interactive map visualization
- AI-based severity prediction
- Real-time alerts and notifications
- Admin dashboard for incident management

## 🛠️ Tech Stack
- **Frontend:** React.js, Bootstrap
- **Backend:** Spring Boot, Spring Security
- **Database:** MySQL
- **AI:** Python, Scikit-learn
- **Maps:** Google Maps API / Leaflet

## ⚙️ Installation

```bash
# Backend
cd backend
mvn spring-boot:run

# Frontend
cd frontend
npm install
npm start
```
